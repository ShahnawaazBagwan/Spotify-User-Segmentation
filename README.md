What if Spotify could understand listeners without asking them anything?

I recently done a Spotify User Segmentation using Unsupervised Machine Learning — where models discover patterns without labeled data.

Instead of predicting outcomes, I asked:

 Can we group users purely based on listening behavior?

What I explored
K-Means Clustering
Gaussian Mixture Models (Full, Tied, Diagonal, Spherical)
Multiple scaling techniques to improve clustering performance

After experimenting with several iterations and evaluations, I successfully identified 4 unique listener personas:

🎵 Loyal Listeners
🎵 Casual Explorers
🎵 Power Users
🎵 Passive Users

Why this matters
User segmentation like this helps platforms:
• Improve music recommendations
• Target premium subscriptions
• Reduce user churn
• Personalize engagement strategies

Biggest Learning:
In Unsupervised ML, data preparation & scaling matter more than the algorithm itself.

This project strengthened my understanding of:
Machine Learning | Clustering | Feature Engineering | Business Interpretation
